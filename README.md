# ApiFox Security Toolkit

ApiFox 2026年3月供应链投毒事件的检测、评估与修复工具集。

## 功能规划

- 检测本机是否受影响
- 评估泄露资产范围
- 自动轮换 SSH 密钥
- 自动轮换 Git 凭证
- K8s 凭证处置建议
- 生成安全报告
